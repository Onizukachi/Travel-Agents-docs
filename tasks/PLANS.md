Перед тем как писать код, создай в папке `.tasks` новый файл `task-{следующий номер таски}` и укажи в нем:

1. **План архитектуры** - стек технологий, ключевые решения и их обоснование
2. **Структурированный список задач**
   Разбей реализацию на минимальные атомарные шаги, которые можно выполнить без дополнительного ввода
   - [ ] Шаг 1: ...
   - [ ] Шаг 2: ...
3. **Заметки для восстановления сессии**
   Добавь подробные заметки о:
   - Текущем состоянии проекта и уже выполненной работе
   - Критически важных принятых решениях
   - Точке, с которой нужно продолжить, если сессия прервется

Пиши конкретно и применимо на практике. Этот план будет направлять всю реализацию.
Перед стартом задай вопросы по любым пунктам, которые в плане описаны недостаточно конкретно.
